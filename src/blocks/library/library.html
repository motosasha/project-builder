Библиотека компонентов

  • button

    Кнопка. Универсальный класс который может использоваться для button, input(type=”button”) и ссылок

  • input

    Поле ввода. Текст, почта, телефон, поиск, пароль, число. У поля отслеживается состояние

  • textarea

    Текстовое поле. У поля отслеживается состояние, а также подключено автоизменение высоты

  • select

    Поле выбора. Выпадающее меню системное

  • checkbox

    Чекбокс

  • radio

    Радио-кнопка

  • field

    Поле формы. Лейбл, информация об ошибке. Поле формы может содержать поле ввода, текстовое поле, поле выбора

    Текст ошибки

  • field-dp

    Поле формы. Лейбл, информация об ошибке. С анимированным лейблом-плейсхолдером

    Текст ошибки
